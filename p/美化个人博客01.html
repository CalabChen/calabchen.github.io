给hugo 个人博客加入 aplayer音乐播放器并且实现全站播放(跳转页面时音乐记录不会被打断)

May 11, 2024
阅读时长: 3 分钟
1312 字

前言

对于hugo-theme-stack主题，网上有很多人都有写如何添加音乐播放器，但是我全都读完之后，许多文章在最后的配置里写的不明白，让人费解。我出这篇文章的目的就是集他人之长处，把整个过程给大家呈现出来，方便大家都搭建成功，让大家为美化的自己的主题越来越充满信心，获得满足感和成就感。如果对我的配置不满足，那就请各位加油努力自己改造了。成功的话，可以在文章的评论区里展示一下成果，甚至和我一样写一篇博客，发在评论区去帮助更多的人！ 话不多说，请看下面的文章。

思路

如果你看到关于hugo的shortcodes来实现音乐播放器的文章,抱歉的告诉你，那是在markdown文章里嵌入音乐播放器(或者说是一个可以点击的图形化链接)。而我希望实现的是全局的音乐播放器，无论跳转页面都有音乐播放，我的实现方法是在\layouts\partials\footer里新建一个名字为aplayer.html的文件，这样可以在网站的页脚实现音乐播放器，并且以我的审美来看不会影响网站的观感。

创建完了上面的文件后，我们打开同一目录(即\layouts\partials\footer)下的include.html文件，在最后加入下面的代码:

1
2

<!-- 添加aplayer音乐播放器 -->
{{ partial "footer/aplayer.html" . }}

这里有点类似于程序员写代码时引用的头文件，不加的话网站就会报错或者网站忽略新建的文件(因为它不知道你要干嘛，你忙活半天就是徒劳一场)

关闭include.html文件之后打开aplayer.html文件。

添加Apalyer音乐播放器

在空空的文件里加入下面的代码，添加Aplayer插件:

<!DOCTYPE html>
<html>

<head>
    <!-- require APlayer -->
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/aplayer/dist/APlayer.min.css">
    <script src="https://cdn.jsdelivr.net/npm/aplayer/dist/APlayer.min.js"></script>   
</head>

添加MetingJS

接着，继续在<head></head>添加下面的代码, 就是加到Aplayer插件后面:

1
2

 <!-- require MetingJS -->
    <script src="https://cdn.jsdelivr.net/npm/meting@2.0.1/dist/Meting.min.js"></script>

添加pjax

然后,继续在<head></head>添加下面的代码, 就是加到Meting插件后面:

 <!-- require pjax -->
    <script src="https://cdn.bootcss.com/jquery/3.4.1/jquery.min.js"></script>
    <script src="https://cdn.bootcss.com/jquery.pjax/2.0.1/jquery.pjax.min.js"></script>
    
    <script>
        // 对所有链接跳转事件绑定pjax容器container
        $(document).pjax('a[target!=_blank]', '#container', { fragment: '#container', timeout: 8000 })
    </script>

好嘞，基本工作完成，让我们进行使用配置吧。

<body>
    <div class="demo">
        <div id="player1">

    <script>
        var ap = new APlayer
            ({
                element: document.getElementById('player1'),
                fixed: true,
                autoplay: false,
                mini: true,
                theme: '#f8f4fc',
                loop: 'all',
                order: 'list',
                preload: 'auto',
                volume: 0.5,
                mutex: true,
                listFolded: true,
                listMaxHeight: '500px',
                lrcType: 0,
                music: [
                    {
                        name: 'name',
                        artist: 'artist',
                        url: 'url.mp3',
                        cover: 'cover.jpg'
                    }
                ]

        //ap.init();
    </script>

     <!-- MetingJS使用-->
    <meting-js 
        fixed="true" 
        server="netease"  
        type="playlist" 
        id="60198">
    </meting-js>

    <!-- APlayer 跳转页面保持音乐播放进度， 下面的代码都是实现这个功能 -->
    <script>
        ap = null
        Object.defineProperty(document.querySelector('meting-js'), "aplayer", {
            set: function (aplayer) {
                ap = aplayer
                ready();
            }
        });

        isRecover = false;
        function ready() {
            ap.on('canplay', function () {
                if (!isRecover) {
                    if (localStorage.getItem("musicIndex") != null) {
                        musicIndex = localStorage.getItem("musicIndex");
                        musicTime = localStorage.getItem("musicTime");
                        if (ap.list.index != musicIndex) {
                            ap.list.switch(musicIndex);
                        } else {
                            ap.seek(musicTime);
                            ap.play();
                            localStorage.clear();
                            isRecover = true;
                        }
                    } else {
                        isRecover = true;
                    }
                }
            });
        }

        window.onbeforeunload = function (event) {
            if (!ap.audio.paused) {
                musicIndex = ap.list.index;
                musicTime = ap.audio.currentTime;
                localStorage.setItem("musicIndex", musicIndex);
                localStorage.setItem("musicTime", musicTime);
            }
        };
    </script>
</body>

好了，启动或者调试中的网站就会在左下角展示一个音乐播放器，大功告成啦！
